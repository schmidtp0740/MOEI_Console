# MOEI Console

## Prereqs
- have node installed

## Setup
```
$ git clone https://github.com/schmidtp0740/MOEI_Console
$ cd ./MOEI_Console
$ npm install
$ npm start
```

- go to localhost:3000 in your favorite browser
